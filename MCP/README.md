# 🔌 Model Context Protocol (MCP) Implementations [Work in progress]

This folder contains my work with the **Model Context Protocol**, focusing on creating secure interfaces between LLMs and local/remote resources.

## 🚀 Servers Developed

- **Custom SQLite Server:** Allowing agents to query local databases safely.
- **Filesystem Tool:** Enabling controlled file read/write operations for AI agents.
- **API Bridge:** Connecting Claude/GPT to proprietary external APIs.

## 🛠️ Stack

- **Languages:** Python (Fastmcp), Node.js (TypeScript).
- **Clients:** Claude Desktop, VS Code.

## 🎯 Objective

To build a seamless ecosystem where AI models have the right "context" and "tools" to act on real-world data without compromising security.
